# OEM Grafico Simple Excel - Escala Técnica Fija

Cambios aplicados sobre el app.py original:
- Se conserva carga de Excel una vez.
- Se conserva selección de hoja/cilindro.
- Se conserva selección UPPER/LOWER.
- Se conserva diseño, tabla, KPIs, PNG y PDF.
- Solo se cambia la escala automática por escala técnica fija.

Escala:
- <0.90 mm: desgaste severo
- 0.90-0.99 mm: desgaste importante
- 1.00-1.09 mm: seguimiento
- 1.10-1.19 mm: normal
- >=1.20 mm: excelente

Main file path: app.py
