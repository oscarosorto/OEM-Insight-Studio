# OEM Insight Studio Web v1.0

Aplicación web piloto para generar reportes visuales de inspección de Big End Bearing.

## Uso en Streamlit Cloud

1. Crear repositorio en GitHub.
2. Subir estos archivos:
   - app.py
   - requirements.txt
   - carpeta assets
3. Entrar a https://streamlit.io/cloud
4. Iniciar sesión con GitHub.
5. Crear nueva app.
6. Seleccionar el repositorio.
7. Main file path: app.py
8. Deploy.

## Funciones

- Cargar Excel Banco A.
- Cargar Excel Banco B.
- Seleccionar cilindros.
- Generar reporte visual tipo WSM.
- Descargar PNG y PDF.

## Nota

La versión piloto está ajustada a la estructura de los archivos:
- BIG END BEARING BANCO A GENSET 11
- BIG END BEARING BANCO B GENSET 11
